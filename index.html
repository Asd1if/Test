Квадратные уравнения

Квадратное уравнение — алгебраическое уравнение второй степени общего вида:

ax² + bx + c = 0, где a ≠ 0

Дискриминант: определяет количество и тип корней:

D = b² - 4ac

Корни уравнения:

x₁,₂ = (-b ± √D) / (2a)

Свойства дискриминанта:

  • D > 0: два различных действительных корня
  • D = 0: один корень (кратный)
  • D < 0: два комплексно-сопряженных корня

Теорема Виета: связь между корнями и коэффициентами:

x₁ + x₂ = -b/a
x₁ × x₂ = c/a

import math

def solve_quadratic(a, b, c):
    """Решает квадратное уравнение ax² + bx + c = 0"""
    discriminant = b**2 - 4*a*c
    
    if discriminant > 0:
        x1 = (-b + math.sqrt(discriminant)) / (2*a)
        x2 = (-b - math.sqrt(discriminant)) / (2*a)
        return f"Два корня: x₁ = {x1:.2f}, x₂ = {x2:.2f}"
    elif discriminant == 0:
        x = -b / (2*a)
        return f"Один корень: x = {x:.2f}"
    else:
        real_part = -b / (2*a)
        imaginary_part = math.sqrt(-discriminant) / (2*a)
        return f"Комплексные корни: {real_part:.2f} ± {imaginary_part:.2f}i"

# Пример использования
print(solve_quadratic(1, -3, 2))  # x² - 3x + 2 = 0
print(solve_quadratic(1, -4, 4))  # x² - 4x + 4 = 0
print(solve_quadratic(1, 2, 5))   # x² + 2x + 5 = 0

Прогрессии

Арифметическая прогрессия: последовательность чисел, в которой разность между последующим и предыдущим членами постоянна.

aₙ = a₁ + d(n-1)
где d — разность прогрессии

Сумма n членов арифметической прогрессии:

Sₙ = n/2 × (a₁ + aₙ) = n/2 × [2a₁ + d(n-1)]

Геометрическая прогрессия: последовательность чисел, в которой отношение последующего члена к предыдущему постоянно.

bₙ = b₁ × qⁿ⁻¹
где q — знаменатель прогрессии (q ≠ 0)

Сумма n членов геометрической прогрессии:

Sₙ = b₁ × (qⁿ - 1)/(q - 1) при q ≠ 1
Sₙ = n × b₁ при q = 1

Сумма бесконечно убывающей геометрической прогрессии (|q| < 1):

S = b₁ / (1 - q)

Теорема Пифагора

Теорема Пифагора — фундаментальное утверждение евклидовой геометрии, устанавливающее соотношение между сторонами прямоугольного треугольника.

c² = a² + b²
где c — гипотенуза, a и b — катеты

Доказательства:

  1. Через площади квадратов: Самый известный метод, приписываемый Пифагору
  2. Метод подобных треугольников: Использует свойства подобных фигур
  3. Алгебраическое доказательство: Через алгебраические преобразования
  4. Доказательство Евклида: Из "Начал" Евклида

Обратная теорема: Если квадрат одной стороны треугольника равен сумме квадратов двух других сторон, то треугольник прямоугольный.

Применение:

  • Вычисление расстояний в декартовой системе координат
  • Определение длины диагонали прямоугольника
  • Расчеты в строительстве и архитектуре
  • Тригонометрия: основа для определения тригонометрических функций
Python 
import math

def pythagorean_theorem(a, b, find='c'):
    """
    Применяет теорему Пифагора
    find: 'c' - найти гипотенузу, 'a' или 'b' - найти катет
    """
    if find == 'c':
        # Находим гипотенузу
        return math.sqrt(a**2 + b**2)
    elif find == 'a':
        # Находим катет a, зная гипотенузу b и катет c
        return math.sqrt(b**2 - a**2)
    elif find == 'b':
        # Находим катет b, зная гипотенузу a и катет c
        return math.sqrt(a**2 - b**2)
    else:
        return "Неверный параметр find"

def distance_between_points(x1, y1, x2, y2):
    """Расстояние между двумя точками на плоскости"""
    return math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

# Примеры использования
print(f"Гипотенуза при катетах 3 и 4: {pythagorean_theorem(3, 4, 'c'):.2f}")
print(f"Катет при гипотенузе 5 и катете 3: {pythagorean_theorem(3, 5, 'a'):.2f}")
print(f"Расстояние между (0,0) и (3,4): {distance_between_points(0, 0, 3, 4):.2f}")

# Проверка на прямоугольность треугольника
def is_right_triangle(a, b, c):
    """Проверяет, является ли треугольник прямоугольным"""
    sides = sorted([a, b, c])
    return math.isclose(sides[2]**2, sides[0]**2 + sides[1]**2)

print(f"Треугольник со сторонами 3,4,5 прямоугольный: {is_right_triangle(3, 4, 5)}")
print(f"Треугольник со сторонами 5,6,7 прямоугольный: {is_right_triangle(5, 6, 7)}")

Основные тригонометрические функции

Тригонометрия изучает соотношения между сторонами и углами треугольников.

Основные функции для прямоугольного треугольника:

sin α = противолежащий катет / гипотенуза
cos α = прилежащий катет / гипотенуза
tan α = sin α / cos α = противолежащий / прилежащий
cot α = cos α / sin α = прилежащий / противолежащий

Основное тригонометрическое тождество:

sin² α + cos² α = 1

Формулы приведения:

sin(90° - α) = cos α
cos(90° - α) = sin α
sin(180° - α) = sin α
cos(180° - α) = -cos α

Законы Ньютона

Первый закон (закон инерции): Существуют такие системы отсчёта, называемые инерциальными, относительно которых тело сохраняет состояние покоя или равномерного прямолинейного движения, если на него не действуют другие тела или их действие скомпенсировано.

Если ΣF = 0, то a = 0

Второй закон (основной закон динамики): Ускорение тела прямо пропорционально равнодействующей сил, приложенных к телу, и обратно пропорционально его массе.

F = m × a
где F — сила (Н), m — масса (кг), a — ускорение (м/с²)

Третий закон (закон действия и противодействия): Силы, с которыми тела действуют друг на друга, равны по модулю и противоположны по направлению.

F₁₂ = -F₂₁

Сила тяжести:

F = m × g
где g ≈ 9.81 м/с² — ускорение свободного падения
Python 
import math

class PhysicsCalculator:
    G = 9.81  # ускорение свободного падения, м/с²
    
    @staticmethod
    def newtons_second_law(mass, acceleration):
        """Второй закон Ньютона: F = m * a"""
        return mass * acceleration
    
    @staticmethod
    def weight(mass):
        """Сила тяжести: F = m * g"""
        return mass * PhysicsCalculator.G
    
    @staticmethod
    def free_fall_time(height):
        """Время свободного падения с высоты h"""
        return math.sqrt(2 * height / PhysicsCalculator.G)
    
    @staticmethod
    def free_fall_velocity(time):
        """Скорость после времени t свободного падения"""
        return PhysicsCalculator.G * time

# Примеры расчетов
mass = 10  # кг
acceleration = 2  # м/с²

force = PhysicsCalculator.newtons_second_law(mass, acceleration)
print(f"Сила при массе {mass} кг и ускорении {acceleration} м/с²: {force} Н")

weight = PhysicsCalculator.weight(mass)
print(f"Вес тела массой {mass} кг: {weight} Н")

height = 100  # метров
fall_time = PhysicsCalculator.free_fall_time(height)
velocity = PhysicsCalculator.free_fall_velocity(fall_time)
print(f"Время падения с {height} м: {fall_time:.2f} с")
print(f"Скорость в момент удара: {velocity:.2f} м/с ({velocity * 3.6:.2f} км/ч)")

Законы термодинамики

Первый закон (закон сохранения энергии): Изменение внутренней энергии системы равно сумме количества теплоты, переданной системе, и работы, совершённой над системой.

ΔU = Q - A
где ΔU — изменение внутренней энергии,
Q — количество теплоты,
A — работа, совершённая системой

Второй закон: Невозможен процесс, единственным результатом которого является передача тепла от менее нагретого тела к более нагретому.

Третий закон (теорема Нернста): При приближении температуры к абсолютному нулю энтропия любой системы стремится к постоянному значению.

Законы Менделя

Основные законы наследственности, открытые Грегором Менделем в 1865 году.

Первый закон (закон единообразия гибридов первого поколения): При скрещивании двух гомозиготных организмов, отличающихся по одной паре альтернативных признаков, всё первое поколение гибридов окажется единообразным и будет нести признак одного из родителей.

Второй закон (закон расщепления): При скрещивании гибридов первого поколения между собой во втором поколении наблюдается расщепление в определённом числовом отношении: 3:1 по фенотипу, 1:2:1 по генотипу.

Третий закон (закон независимого наследования): При дигибридном скрещивании у гибридов каждая пара признаков наследуется независимо от других и даёт расщепление 3:1, образуя при этом 4 фенотипические группы в соотношении 9:3:3:1.

import itertools

class Genetics:
    @staticmethod
    def monohybrid_cross(parent1, parent2):
        """
        Моногибридное скрещивание
        parent1, parent2: генотипы родителей (например, 'AA', 'Aa', 'aa')
        """
        # Генерация гамет
        def get_gametes(genotype):
            return list(set(genotype[i] for i in range(len(genotype))))
        
        gametes1 = get_gametes(parent1)
        gametes2 = get_gametes(parent2)
        
        # Все возможные комбинации
        offspring = []
        for g1 in gametes1:
            for g2 in gametes2:
                # Сортировка для единообразия (A перед a)
                genotype = ''.join(sorted(g1 + g2))
                offspring.append(genotype)
        
        # Подсчет результатов
        result = {}
        for genotype in offspring:
            result[genotype] = result.get(genotype, 0) + 1
        
        total = len(offspring)
        ratios = {k: f"{v}/{total}" for k, v in result.items()}
        
        return {
            'gametes_parent1': gametes1,
            'gametes_parent2': gametes2,
            'offspring': offspring,
            'ratios': ratios
        }
    
    @staticmethod
    def predict_phenotype(genotype, dominant_trait='A', recessive_trait='a'):
        """Предсказание фенотипа по генотипу"""
        if dominant_trait in genotype:
            return "Доминантный признак"
        else:
            return "Рецессивный признак"

# Пример использования
print("=== Моногибридное скрещивание ===")
cross_result = Genetics.monohybrid_cross('Aa', 'Aa')
print(f"Гаметы родителя 1: {cross_result['gametes_parent1']}")
print(f"Гаметы родителя 2: {cross_result['gametes_parent2']}")
print(f"Потомство: {cross_result['offspring']}")
print(f"Соотношения генотипов: {cross_result['ratios']}")

# Фенотипическое расщепление
offspring = cross_result['offspring']
dominant_count = sum(1 for g in offspring if 'A' in g)
recessive_count = sum(1 for g in offspring if g == 'aa')
print(f"\nФенотипическое расщепление: {dominant_count}:{recessive_count} (3:1)")

# Пример для разных родителей
print("\n=== Скрещивание AA × aa ===")
cross2 = Genetics.monohybrid_cross('AA', 'aa')
print(f"Все потомки: {cross2['offspring']}")
print(f"Фенотип всех потомков: {Genetics.predict_phenotype('Aa')}")
